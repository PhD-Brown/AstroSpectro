# <FontAwesomeIcon icon={faVial} /> Contexte Scientifique

Bienvenue dans la section scientifique du projet **Astro-Spectro**.

Cette partie vous permettra de comprendre les fondations scientifiques qui motivent la conception du pipeline. En allant au-delà de l’aspect technique, vous découvrirez :
- L’origine et les caractéristiques des données spectroscopiques utilisées,
- La méthodologie de traitement des spectres jusqu’à leur classification,
- La manière dont on évalue les performances des modèles et interprète les sorties.

Que vous soyez curieux d’un point de vue astrophysique ou que vous souhaitiez approfondir la logique du pipeline, cette section vous donnera toutes les clés.

<br />

:::info[<FontAwesomeIcon icon={faUserGraduate} /> Pour qui ?]
Cette section est idéale si :
- Vous débutez en astrophysique computationnelle,
- Vous voulez comprendre l’origine et le traitement des spectres LAMOST,
- Vous cherchez à relier les sorties du modèle à leur signification scientifique.
:::
