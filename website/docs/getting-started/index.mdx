# <FontAwesomeIcon icon={faRocket} /> Bienvenue dans le Démarrage Rapide

<br />

Cette série de pages vous guide étape par étape pour préparer votre environnement local et lancer une première classification spectrale complète en quelques minutes. 

> L'objectif est de vous amener à faire fonctionner **le pipeline complet**, de l'installation à l'obtention de vos premières prédictions de classes stellaires, sans avoir à comprendre tous les détails internes (<FontAwesomeIcon icon={faUserAstronaut} /> pour l’instant).
