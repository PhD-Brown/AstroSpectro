# <FontAwesomeIcon icon={faRocket} /> Bienvenue dans le Démarrage Rapide

<br />

Cette série de pages vous guide étape par étape pour préparer votre environnement local et lancer une première classification spectrale complète en quelques minutes. 

> L'objectif est de vous amener à faire fonctionner **le pipeline complet**, de l'installation à l'obtention de vos premières prédictions de classes stellaires, sans avoir à comprendre tous les détails internes (pour l’instant 🧑‍🚀).

<FontAwesomeIcon icon="fa-solid fa-rocket" />

<br />

## Itinéraire Suggéré

Voici l’ordre recommandé pour progresser sans accroc :

:::note[ÉTAPE 1 : INSTALLATION]
<FontAwesomeIcon icon={faDownload} size="2x" pull="left" />

La première étape consiste à préparer votre environnement Python et à télécharger les dépendances nécessaires au projet. C'est le fondement de tout le reste.

<br/>

<a className="button button--primary" href="./installation">
  Aller au guide d’installation →
</a>
:::

<br />

:::note[ÉTAPE 2 : PREMIER LANCEMENT]
<FontAwesomeIcon icon={faCodeBranch} size="2x" pull="left" />

Une fois l’installation terminée, ce guide vous accompagnera pour un lancement complet qui va :

-   Télécharger automatiquement des spectres LAMOST DR5,
-   Extraire les features principales,
-   Entraîner un modèle de classification simple, et
-   Afficher les résultats.

<br/>

<a className="button button--primary" href="./first-run">
  Lancer le pipeline →
</a>
