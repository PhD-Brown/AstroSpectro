# <FontAwesomeIcon icon={faQuoteRight} /> Citer AstroSpectro

Merci de considérer l'utilisation de ce projet dans vos travaux de recherche. Si vous utilisez **AstroSpectro**, nous vous serions reconnaissants de bien vouloir le citer. La citation académique est essentielle pour la reconnaissance et la pérennité des logiciels open-source en science.

## <FontAwesomeIcon icon={faBook} /> Format Préféré (BibTeX)

Pour toute publication scientifique, le format BibTeX est le plus approprié. Vous pouvez copier-coller l'entrée suivante dans votre fichier `.bib`.

<Admonition type="info" title="Entrée BibTeX">

```bibtex
@misc{baker2025astrospectro,
  author       = {Baker, Alex},
  title        = {{AstroSpectro: A Pipeline for Automated Stellar Spectral Classification}},
  year         = {2025},
  publisher    = {GitHub},
  journal      = {GitHub repository},
  howpublished = {\url{https://github.com/PhD-Brown/AstroSpectro}}
}
```

</Admonition>

:::note Bientôt : un DOI via Zenodo !
Nous prévoyons de créer une archive officielle de ce projet sur Zenodo pour la sortie de la version 1.0. Cela fournira un DOI (Digital Object Identifier), qui est une référence permanente et encore plus stable pour la citation de logiciels.
:::

### Citation simple
Pour une référence rapide dans un rapport, un article de blog ou autre, vous pouvez utiliser le format texte suivant :

```Baker, A. (2025). AstroSpectro: A Pipeline for Automated Stellar Spectral Classification. [Software]. Disponible sur : https://github.com/PhD-Brown/AstroSpectro```
